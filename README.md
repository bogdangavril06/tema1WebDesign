#Website Portofoliu

Acest proiect prezinta un scurt portofoliu personal

#Functionalitati

- 4 pagini
- meniu de navigatie
- fisier style.css separat pentru un layout mai friendly
- tabel stilizat pentru easier reading
- efect hover
- structura semantica curata

#Limitatii

- design simplu dar poate fi imbunatatit
- fara animatii complexe
- formularul este "fictiv" - nu functioneaza real
- tabelul este hardcoded - site-ul nu are o baza de date deci nu poate fi updatat direct, orice schimbare trebuie sa fie codata.
